####  
## 03. 조건에 맞는 데이터 가져오기 
#### ► [03_where_221010]  
- true 조건에 맞는 데이터를 가져오는 WHERE 실습
- 비교 연산자(=, !=, >, >=, <, <=) / 논리 연산자(NOT, OR, NOT) / 기타 주요 연산자(BETWEEN, IN 등)의 이해 및 일부 실습
- 와일드 카드(%, _)를 사용해 특정 문자열이 포함된 데이터를 선택하는 LIKE 실습
- NULL 데이터를 다루는 IS NULL 실습
####
- ✔ 와일드카드 '%'는 0개 이상의 문자 의미하고, '_'는 1개의 문자를 의미
